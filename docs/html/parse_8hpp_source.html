279  Rules 'ABC' + 'XYZ' maybe defined expecting the two terminals be separated by whitespace but this algorithm could parse the string 'ABCXYZ' as two separate terminals.
280  There's a number of traditional approaches to solving this such as tokenizing before parsing or creating parse tables.

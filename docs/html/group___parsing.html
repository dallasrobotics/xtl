class xtd::parse::whitespace< _Ch >

list of whitespace characters to ignore in the input stream.

Whitespace characters are parsed and discarded from the input stream when encountered between rules or terminals. A typical whitespace declaration for written language might be: whitespace<'', '
', '', ' '>;
